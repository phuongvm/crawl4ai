# Suggested Commands

## Setup & dependencies
- **Install Dependencies**: `uv sync` or `pip install -e .`
- **Install Playwright Browsers**: `playwright install` or `python -m playwright install`
- **Post-Install Setup**: `crawl4ai-setup`

## Development
- **Run CLI**: `python -m crawl4ai.cli` or `crwl`
- **Run a Script**: `python path/to/script.py`

## Testing
- **Run All Tests**: `pytest`
- **Run Specific Test File**: `pytest tests/test_file.py`
- **Run with Output**: `pytest -s`

## Documentation
- **Serve Docs Locally**: `mkdocs serve`
- **Build Docs**: `mkdocs build`

## Linting & Formatting (Recommended)
- **Format Code**: `ruff format .`
- **Check Linting**: `ruff check .`
